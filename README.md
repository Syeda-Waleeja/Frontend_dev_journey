# Web Dev Journey

A complete web development journey featuring projects, experiments, UI practice, animations, backend practice, and daily progress tracking.

---

## 📖 About This Repository

This repository is a structured space for learning and improving web development skills through consistent practice, real projects, experiments, UI recreations, and development logs.

The goal is not only to build projects, but also to maintain discipline, improve problem-solving skills, and document growth as a developer.

## 📂 Folder Details

### projects/
Complete applications and portfolio-level projects.

### experiments/
Small learning tests, feature experiments, and practice components.

### ui-clones/
Frontend recreations of popular website interfaces for UI/UX practice.

### animations/
CSS and JavaScript animations, transitions, and interaction effects.

### backend/
Backend learning, APIs, authentication systems, databases, and server-side practice.

### notes/
Personal learning notes, concepts, references, and revision material.

### challenges/
Solutions to coding challenges from different practice platforms.

### dev-log/
Daily and weekly development progress tracking.
